UCDxml.CodePointsSet — Type
abstract type CodePointsSet end

There are two types of code point representation. SingleCodePoint and RangeCodePoint with first and last codepoint. Each codepoint value is UInt32 (as it is expressed as hexadecimal number with four to six digits, ie. minimum is 0x0000 and maximum is 0xFFFFFF(UAX#44 #4.2.2))
